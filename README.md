Swaggar document: [https://witamy.pythonanywhere.com](https://witnamy.pythonanywhere.com)

## Initial setup

```
cd /home/download/project/witamy-server
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install -r requirement.txt
chmod u+x run.sh
./run.sh
```
